# {{project_name}} Architecture Knowledge Starter

This folder is generated by `akdb setup`.

Edit the starter files, then re-import them whenever the architecture memory should be refreshed:

```powershell
akdb adr import --project {{project_id}} --folder adr
akdb uml import --project {{project_id}} --folder uml
```

Recommended first pass:

1. Replace placeholder names in `specs/ARCHITECTURE_SPEC.md`.
2. Update the ADR decisions in `adr/`.
3. Update the diagrams in `uml/`.
4. Run `akdb consistency check --project {{project_id}}`.
