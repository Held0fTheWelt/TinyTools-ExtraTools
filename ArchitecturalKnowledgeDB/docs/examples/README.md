# Examples

This folder contains small AKDB examples and fixtures.

| Path | Purpose |
| --- | --- |
| `architectural-knowledge-db.projects.yaml` | Example project registry. |
| `docker-compose.yml` | Example compose setup for local service use. |
| `standalone-sample/` | Minimal sample project with docs, ADR, knowledge JSON, and diagram input. |

Examples should stay small and self-contained. Do not copy real external repositories or generated Tiny Tools exports here.
