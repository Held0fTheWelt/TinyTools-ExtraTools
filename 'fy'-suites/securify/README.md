# Securify

Securify is the fy suite responsible for outward-facing repository security hygiene.
It scans repositories for likely secret exposure, missing security-facing guidance,
and weak scope protections that should be visible before work continues.

Securify is a core fy suite. It complements:
- `contractify` by connecting security guidance to explicit governance,
- `testify` by exposing security-related follow-up needs,
- `documentify` by surfacing missing security guidance,
- `docify` by highlighting places where security-sensitive code needs better explanation,
- `usabilify` by ensuring security guidance remains understandable to humans.
